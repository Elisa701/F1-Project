About This Project

This website was created as part of the Web Development course project for the academic year 2026. The goal was to design and implement a complete, multi‑page website using only HTML and CSS, while applying key concepts such as typography, layout, accessibility, responsive design, and performance optimization.

Why Formula 1?

Formula 1 has always fascinated me for its combination of speed, technology, strategy, and human skill. Choosing this theme allowed me to work with rich visual content and structured information, while creating a website that is both engaging and educational.

Project Structure

The website is organized into several sections, each dedicated to a specific aspect of the 2026 Formula 1 season:

  • Teams overview and individual team pages
  • Drivers and team principals profiles
  • Circuits and track details
  • Season grid summary using CSS Grid
  • Privacy page following course guidelines

Technologies Used

The project was developed using:

  • Semantic HTML5
  • Custom CSS3 (no frameworks)
  • CSS Grid and Flexbox for layout
  • Responsive design with media queries
  • Accessibility improvements (ARIA labels, roles, alt text, keyboard navigation)
  • Performance enhancements such as lazy loading for images

What I Learned

Throughout the development of this project, I strengthened my understanding of web structure, layout techniques, and responsive design. I also learned how to maintain consistency across multiple pages, how to improve accessibility, and how to create a visually coherent theme using custom CSS.

Future Improvements

If expanded in the future, the project could include interactive elements using JavaScript, dynamic data updates, or additional pages such as race results, standings, or historical statistics.

This project represents a complete and functional demonstration of the skills acquired during the course, and it reflects both my passion for Formula 1 and my growth as a web developer.
